# Web3 Payroll Streaming

A production-grade Web3 payroll platform that enables real-time salary streaming using smart contracts.

## Overview

This system allows employers to stream salaries continuously, enabling employees to withdraw earnings at any time instead of waiting for fixed payroll cycles.

## Features

- Real-time salary streaming
- Withdraw anytime functionality
- Role-based dashboards (Employer / Employee)
- ERC20 token payments
- Pause and resume streams
- Event-driven backend indexing
- Analytics dashboard

## Tech Stack

- Smart Contracts: Solidity (Foundry)
- Backend: Node.js + Express + MongoDB
- Frontend: React + Tailwind
- Web3: ethers.js

## Architecture

- Smart contracts handle payroll logic
- Backend indexes blockchain events
- Frontend interacts via APIs and wallet
